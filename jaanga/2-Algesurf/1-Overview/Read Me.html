On the following pages you have access to apps that can help you explore many different equations. And then once you are happy with the resulting visualization, you can save it, create a link to it, embed it in a web page and so on.

To help you, there are a number of galleries of equations and their visualizations.

Features

  • Free and open source software with a flexible license
  • Hardware accelerated graphics in your broeswer with no plugin or download necessary
  • Fully operational in the browser - no server required
  • Generally, JavaScript library agnostic
  • Realtime user interaction with zoom, pan and rotate
  • Full RESTful API
  • Fast and easy editing of equations
  • Robust equation parser with error messaging
  • Support for a variety of materials, textures and shaders
  • Ability to export and save data in variety of ways
